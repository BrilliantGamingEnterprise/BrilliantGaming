Brilliant Gaming — GA4 + Google Search 优化补丁
================================================

基于 GitHub 最新版本：cfac436（时间+购物车更新）
GA4 衡量 ID：G-MG9CK2YDHG

安装方法
--------
1. 解压此压缩包。
2. 将压缩包内除了“安装说明.txt”以外的全部文件，复制到 VS Code 的 BrilliantGaming 项目最外层。
3. 出现同名文件提示时，选择“替换/覆盖”。
4. analytics.js、seo.js、sitemap.xml 是新文件，也必须一起复制进去。
5. 在 VS Code 上传（Commit + Sync/Push）到 GitHub。
6. 等待 Netlify 自动完成部署。

上线后需要做的一次操作
----------------------
1. 打开 Google Search Console。
2. 选择 brilliantgamingtopup.com。
3. 左侧进入“Sitemap”。
4. 在“添加新的站点地图”输入：sitemap.xml
5. 点击“提交”。

GA4 说明
--------
- 部署完成后会开始收集网站使用数据，Google 后台可能需要数小时至 48 小时显示完整报告。
- 会统计页面浏览、游戏查看、加入购物车、复制订单、语言/币种切换和联系渠道。
- 不会把客户账号、密码、UID、角色名、服务器、订单文字或 WhatsApp 消息发送给 GA4。

本次保留的现有设置
------------------
- 下单时间保持 12 小时制，不显示 (MYT)。
- 商品内容保持“商品名称 x 数量”，不显示 1、2、3 编号和单项价格。
